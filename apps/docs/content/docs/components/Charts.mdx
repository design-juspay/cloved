# Charts Component Documentation

## Description
A comprehensive charting component that supports line, bar, and pie charts with customizable styling and interactive features.

## Features
- Multiple chart types (line, bar, pie)
- Interactive legends with hover effects
- Customizable colors and styling
- Flexible slot system for custom content
- Responsive design
- Custom tooltips
- Axis labeling support

## Props

<BlendTypeTable componentMeta={chartsMeta} />

## Usage Examples

### Basic Line Chart
Simple line chart with data points

```tsx
<Charts
  chartType={ChartType.LINE}
  data={chartData}
  xAxisLabel="Month"
  yAxisLabel="Revenue"
  chartHeaderSlot="Monthly Revenue"
/>
```

### Bar Chart with Custom Colors
Bar chart with custom color scheme

```tsx
<Charts
  chartType={ChartType.BAR}
  data={chartData}
  colors={["#FF6B6B", "#4ECDC4", "#45B7D1"]}
  chartHeaderSlot="Sales by Category"
/>
```

### Pie Chart with Legend
Pie chart with right-positioned legend

```tsx
<Charts
  chartType={ChartType.PIE}
  data={pieData}
  legendPosition={ChartLegendPosition.RIGHT}
  chartHeaderSlot="Market Share"
/>
```
