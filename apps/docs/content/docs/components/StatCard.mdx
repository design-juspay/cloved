# Statcard Component Documentation

## Description
A versatile statistics card component that displays key metrics with various visualization options including charts, progress bars, and change indicators.

## Features
- Multiple display variants (number, line chart, bar chart, progress bar)
- Change indicators with increase/decrease visualization
- Chart data visualization support
- Progress tracking with percentage display
- Icon support for titles and actions
- Help tooltips for additional context
- Responsive design

## Props

<BlendTypeTable componentMeta={statcardMeta} />

## Usage Examples

### Basic Number Card
Simple stat card displaying a number value

```tsx
<StatCard
  title="Total Revenue"
  value="$125,000"
  variant={StatCardVariant.NUMBER}
/>
```

### Card with Change Indicator
Stat card showing positive change

```tsx
<StatCard
  title="Monthly Sales"
  value="$45,000"
  change={{ value: 12, type: ChangeType.INCREASE }}
  variant={StatCardVariant.NUMBER}
/>
```

### Line Chart Card
Stat card with embedded line chart

```tsx
<StatCard
  title="Website Traffic"
  value="12,543"
  variant={StatCardVariant.LINE}
  chartData={[
    { label: "Jan", value: 1200 },
    { label: "Feb", value: 1350 },
    { label: "Mar", value: 1543 }
  ]}
/>
```

### Progress Bar Card
Stat card with progress visualization

```tsx
<StatCard
  title="Goal Progress"
  value="75%"
  variant={StatCardVariant.PROGRESS_BAR}
  progressValue={75}
/>
```
