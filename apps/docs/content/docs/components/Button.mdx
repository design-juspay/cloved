# Button Component Documentation

## Description
A versatile button component with multiple variants, sizes, and states for user interactions.

## Features
- Multiple button types (primary, secondary, danger, success)
- Three sizes (small, medium, large)
- Loading and disabled states
- Icon support (leading and trailing)
- Full width option
- Flexible content alignment
- Button group positioning support

## Props

<BlendTypeTable componentMeta={buttonMeta} />

## Usage Examples

### Basic Button
Simple button with text

```tsx
<Button text="Click me" buttonType={ButtonTypeV2.PRIMARY} />
```

### Button with Icons
Button with leading and trailing icons

```tsx
<Button 
  text="Save" 
  leadingIcon={<SaveIcon />} 
  buttonType={ButtonTypeV2.SUCCESS} 
/>
```

### Loading Button
Button in loading state

```tsx
<Button 
  text="Processing..." 
  loading={true}
  buttonType={ButtonTypeV2.PRIMARY} 
/>
```

### Full Width Button
Button that takes full width of container

```tsx
<Button 
  text="Full Width" 
  fullWidth={true}
  buttonType={ButtonTypeV2.SECONDARY} 
/>
```
